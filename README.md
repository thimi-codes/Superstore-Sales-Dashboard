# Superstore-Sales-Dashboard

## Overview
An interactive sales dashboard built using the Sample Superstore dataset.

## Tools Used
- Python (Pandas) :data cleaning
- Power BI :dashboard and visualizations

## Dataset
Sample Superstore sales data with 10,194 rows covering Orders, Sales, Profit, Region, Category and Sub-Category.

## Key Insights
- West region has the highest sales (739K)
- Tables sub-category is loss-making (negative profit)
- Sales peak in October–November
- Technology leads in both sales and profit

## Project Workflow
1. Raw dataset → cleaned using Python (Pandas) in Google Colab
2. Cleaned CSV exported and loaded into Power BI
3. Interactive dashboard built in Power BI

## Files
- samplesuperstore.csv        — original raw dataset
- super_store_sales.csv      — cleaned dataset
- Superstore_sales_dashboard.ipynb  — data cleaning notebook (Colab)
- superstore_dashboard.pbix   — Power BI dashboard
